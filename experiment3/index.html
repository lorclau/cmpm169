Experiment 3: See That Sound - Sound Art

Description

This piece uses amplitude and frequency analysis to create a spiral visualization of a song and all its fluctuations.

Technical

This code creates a dynamic audio visualizer using p5.js, where visuals react to the frequency spectrum of a song. It loads and plays an audio file, then analyzes the sound's amplitude and frequency spectrum using p5.Amplitude and p5.FFT. As the song plays, the program tracks changes in frequency, and when a significant beat drop is detected (when the average frequency exceeds a threshold), the background color changes to a random color from a predefined set. Simultaneously, spinning rings are drawn at the center of the canvas, with their sizes corresponding to the strength of different frequency bands. These rings change color dynamically based on the spectrum, creating a colorful, rhythmic visual effect. The rotation speed of the rings is controlled by a gradually changing angle, producing a smooth and continuously evolving visual experience synchronized with the music.

Reflection

For this experiment, my main goal was to gain a better understanding of sound art and to try out different ways of representing sound. I believe I gained useful knowledge about sound analysis with p5 through this experiment, so I consider the outcome to be successful. Figuring out how to achieve the spinning rings so that the frequency spiking still worked was a bit of a challenge and took some time to get it looking right. I had never attempted sound visualization with code before, but this presented itself as a great opportunity. While I had to do some more external research compared to previous experiments, I think I learned a lot and I am quite happy with the results. My goal was to create responsive visual art that is both functional and aesthetically pleasing, and I feel like I was able to achieve that with this experiment.
